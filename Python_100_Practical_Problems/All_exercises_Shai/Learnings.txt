3️⃣ How pandas decides what to multiply
Pandas applies operations column-wise, based on data type (dtype).
Numeric columns : int, float → multiply element-wise
Non-numeric columns (strings) : object / string → ignored (left unchanged)

4️⃣ Why strings are NOT duplicated (important!)
In plain Python: "abc" * 2
Output: 'abcabc'

❗ So why doesn’t pandas do this?
Because pandas follows data science semantics, not Python string semantics.
Pandas assumes: “Multiplying a dataset means numeric transformation, not text duplication.”
So pandas skips non-numeric columns.
